# Findings

## Research & Discoveries
- **Polza.ai Unified API**: Предоставляет доступ к 250+ моделям через единый API. Совместим со стандартом OpenAI Chat Completion.
- **Интеграция**: Можно использовать SDK `openai` в Python или Node.js, просто изменив `baseURL`.
- **Модели**: Доступны GPT-4o, Claude, Gemini, Llama и другие.
- **Паттерны копирайтинга**: AIDA (Внимание, Интерес, Желание, Действие), PAS (Проблема, Агитация, Решение), BAB (До, После, Мост).
- **Telegram Bot API**: Для разработки бота рекомендуется использовать `python-telegram-bot`.

## Constraints
1. No self-running commands.
2. Windows environment.
3. Russian language for communication.
